### Prerequisites
- Node.js 18+

- Supabase account

### Installation

```bash
# Clone the repository
git clone https://github.com/Vixcy300/attendance-manager.git

# Navigate to directory
cd attendance-manager

# Install dependencies
npm install

# Create .env file with your credentials
```
